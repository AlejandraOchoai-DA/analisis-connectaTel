# Analisis ConnectaTel
## Objetivo del proyecto
Proyecto de Bootcamp Tripleten en el que se trabajó con información registrada hasta el año 2024, permitiendo analizar el comportamiento del negocio dentro de ese periodo.

## Datasets Utilizados,
1. plans.csv → información de los planes actuales (precio, minutos incluidos, GB incluidos, costo por extra)
2. users.csv → información de los clientes (edad, ciudad, fecha de registro, plan, churn)
3. usage.csv → detalle del uso real de los servicios (llamadas y mensajes)

## Etapas del análisis realizadas
   
Exploramos, limpiamos y se analizaron los datos para construir un perfil estadístico de los clientes, comportamientos atípicos y crear segmentos de clientes.
